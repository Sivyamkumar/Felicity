Why we made Felicity?

The main idea behind the website is to keep you happy and sane. To use the word "mental health" would be big because we understand how difficult it can get. The articles , books and every other thing here is just to make you feel better. However if you feel your mental health detereorating , we highly recommend to seek professional help.

The project solves some of the real-world problems

Javascript , HTML and CSS combined

The coding world today is a lot more intensive and learning a new language/framework can be pretty daunting. But, this doesn't mean we would or could give up. We started teaching ourselves the coding skills needed to create this project and arrived to where we are now. The path was filled with shortcomings, bugs and many more such bad experiences but at the end we were able to solve it with the help of technology.

Technology finds its best use case when it is community driven.

The project started as a mere idea on Instagram and then took its form by conversion into a technical project. Hence, this project not only uses tech stack to tackle some problems but has a bundle of things which aid people in these tough and testing times. Sure, we would have to help our community intensively and as we grow, we will be including much more options which are personal from person to person.

Code by the people, for the people

While learning JS, we came across many obstacles and pieces of puzzles which appeared unsolvable. About the same time, I was diving into Open-source and was amazed by the wonders it offers. The codebase available freely to people makes it a great place for developers and this was the main aim of hosting this project as open source!

This project being an open source project is open for collaboration!
